h1. Installd Client

h2. Description

This repository contains two RubyCocoa Xcode projects :-

* A preference pane
* A status bar

h2. Build

* Install "RubyCocoa":http://rubycocoa.sourceforge.net/
* `gem install choctop`
* `cd mac-prefpane`
* `rake build`
